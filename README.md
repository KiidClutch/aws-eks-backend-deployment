# aws-eks-backend-deployment

End-to-end deployment of a containerized backend application on Amazon EKS using Docker, Amazon ECR, Kubernetes manifests, kubectl, and AWS.

## Project Overview

This project demonstrates how to deploy a containerized backend application to Amazon EKS. The workflow includes:

- Creating an EKS cluster
- Building a Docker image
- Pushing the image to Amazon ECR
- Creating Kubernetes Deployment and Service manifests
- Deploying the application with kubectl
- Verifying the deployment in Amazon EKS
